3 Getting Help

The help system is simple. Type C-h*(or *F1) and fellow the directions. If you are a first-time user, Type C-h t for a tutorial.

3.1 remove help window C-x 1

3.2 scroll help window C-M-v

5 Incremental Search

5.1 search forward C-s

5.2 search backword C-r

5.3 regular expression search C-M-s

5.4 reverse regular expression search C-M-r

5.5 select previous search string M-p

5.6 select next search string M-n

5.7 exit incremental search RET

5.8 undo effect of last character DEL

5.9 abort current search C-g

Use C-s or C-r again to repeat the search in either direction. If Emacs is still searching, C-g cancels only the part not matched.

#+CAPTION Emacs cusor moving

entity to move over backward forward
character C-b C-f
word M-b M-f
line C-p C-n
go to line beginning (or end) C-a C-e
sentence M-a M-e
paragraph M-{ M-}
page C-x [ C-x ]
sexp C-M-b C-M-f
function C-M-a C-M-e
go to buffer beginning (or end) M-< M->
